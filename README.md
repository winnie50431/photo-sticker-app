# 泡泡拍貼機 Photo Sticker!

A Photo Sticker Application created with Processing.

### 【Upload a picture and you can decorate the photo with bubbles and sticker!】

上傳圖片，就可以裝飾照片!

可以選擇貼紙、濾鏡，並點擊產生多彩的動態泡泡!
