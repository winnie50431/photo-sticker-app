# 泡泡拍貼機 Photo Sticker!

### Upload a picture and you can decorate the photo with bubbles and sticker!

上傳圖片，就可以裝飾照片!
裝飾貼紙、選擇濾鏡、也可以隨機產生動態的泡泡!

---

### Responsible for the following functions :

負責工作包含

1. Draw bubble 開發繪製泡泡的功能
2. Paste sticker 貼紙功能
